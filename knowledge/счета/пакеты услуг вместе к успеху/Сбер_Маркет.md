# Пакет РКО «Сбер Маркет»

**Абонентская плата:** 0 / 40 BYN (в зависимости от среднедневных остатков)

**Для кого:** для бизнес-клиентов Сбера: ИП и ЮЛ. Годовая выручка до 10 млн USD. Допустимые виды: оптовая, розничная торговля, ресторанный, гостиничный бизнес.

**Что входит:**
- Безлимит платежей в BYN и инвалюте по Беларуси
- Эквайринг от 1,4% (торговый, мобильный, интернет)
- 2 бизнес-карты (национальная и иностранная валюта)
- Внесение выручки в Белпочте — 0 рублей
- Открытие счетов в инвалюте — 0 рублей

**Тарифы эквайринга:**
- 1,4% по картам Сбера
- 1,9% по картам других банков

**Цифровые сервисы:**
- Проверка контрагента
- Онлайн-бухгалтерия
- Конструктор сайта и управление онлайн-продажами (InSales)
