# Кредит на финансирование затрат по капитальному (текущему) ремонту

- Срок кредита: до 3 лет

**Целевое назначение кредита:**
- Финансирование затрат по капитальному ремонту, в том числе по восстановлению основных физико-технических, эстетических и потребительских качеств основных средств, утраченных в процессе эксплуатации, и/или по улучшению их конструктивных, инженерных, технических, эстетических качеств
- Финансирование затрат по текущему ремонту, предупреждению износа, устранению мелких повреждений и неисправностей, улучшению эстетических качеств основных средств
