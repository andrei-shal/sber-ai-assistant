# Совместная выгода — Партнерская программа

Кредит клиенту под ставку 0,0001% для приобретения продукции, товаров, услуг и объектов основных средств у партнеров Банка

**Возможности для партнеров:**
- Снижение дебиторской задолженности
- Гарантированное получение платежа
- Работа с проверенными покупателями
- Рост объема продаж
- Расширение сети покупателей

**Требования к партнёрам:**
- Резидент Республики Беларусь с текущим счетом в ОАО «Сбер Банк» и Соглашением о сотрудничестве
- Резидент РФ, продающий товары резидентам РБ, с Соглашением о сотрудничестве (счет может быть в любом банке РФ)
- Отсутствие неуплаченных сумм вознаграждений / платежей по Соглашению

**Документы:**
- Условия (оферта) для Партнеров – резидентов РБ
- Перечень документов для Партнера, резидента РБ
- Примерная форма Соглашения с Партнером – резидентом РФ
- Перечень документов для Партнера, резидента РФ
